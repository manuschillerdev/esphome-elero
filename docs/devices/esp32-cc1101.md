# ESP32 + CC1101

Generic ESP32 dev board (ESP32-DevKit, D1 Mini ESP32, etc.) with a CC1101 868 MHz RF module connected over SPI.

## Wiring

| CC1101 | ESP32 | Function |
|--------|-------|----------|
| VCC | 3V3 | Power (3.3V only!) |
| GND | GND | Ground |
| SCK | GPIO18 | SPI clock |
| MOSI | GPIO23 | SPI data out |
| MISO | GPIO19 | SPI data in |
| CSN | GPIO5 | SPI chip select |
| GDO0 | GPIO26 | Interrupt pin |

GDO2 is not used. SPI pins are configurable -- any valid GPIO works.

> **WARNING:** The CC1101 operates at **3.3V**. Never connect it to 5V!

## ESPHome Config

```yaml
esp32:
  board: esp32dev

spi:
  clk_pin: GPIO18
  mosi_pin: GPIO23
  miso_pin: GPIO19

elero:
  cs_pin: GPIO5
  gdo0_pin: GPIO26
```

## Frequency

Most European Elero motors use **868.35 MHz** (the default). If discovery finds nothing, try 868.95 MHz:

```yaml
elero:
  cs_pin: GPIO5
  gdo0_pin: GPIO26
  freq0: 0xc0    # 868.95 MHz (default: 0x7a = 868.35 MHz)
```
